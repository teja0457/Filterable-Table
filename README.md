# Filterable-Table

Filterable-Table with Sorting and Search/Filter Functionality.

Requirements:
- git
- Node v6
- npm v3

To run the app locally:

```bash
git clone https://github.com/teja0457/Filterable-Table.git
cd Filterable-Table
npm run setup
npm run dev
open http://localhost:8080
```
